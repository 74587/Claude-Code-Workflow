# Generate UI Prototypes (/workflow:ui-design:generate)

## Overview
Generate matrix of UI prototypes (`style √ó layout √ó targets`) with production-ready HTML/CSS. Optimized template-based architecture: **`S` times faster** than direct generation.

**Strategy**: Two-layer generation
- **Layer 1**: Generate `L √ó T` reusable templates (agent)
- **Layer 2**: Instantiate `S √ó L √ó T` final prototypes (script)

**Supports**: Pages (full layouts) and components (isolated elements)

## Error Handling

### Common Errors
```
ERROR: No design systems found
‚Üí Run /workflow:ui-design:consolidate first

ERROR: jq not found
‚Üí Install jq: brew install jq

ERROR: Agent task failed
‚Üí Check agent output, retry phase

ERROR: Script permission denied
‚Üí chmod +x ~/.claude/scripts/ui-instantiate-prototypes.sh
```

### Recovery Strategies
- **Partial failure**: Check script logs for counts
- **Missing templates**: Regenerate Phase 4
- **Invalid tokens**: Validate design-tokens.json
